# Security Policy

## Supported Versions

Only the latest version of Hexyn AWS is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to [security@flockyn.com](mailto:security@flockyn.com). All security vulnerabilities will be promptly addressed.

Please do not report security vulnerabilities via GitHub issues.
