Hexyn AWS

A fast, parallelized TUI tool for managing AWS SSM Parameter Store and ECS Task Definitions.

Installation

macOS / Linux
curl -fsSL https://flockyn.github.io/hexyn-aws/install.sh | bash
Windows (PowerShell)
iwr https://flockyn.github.io/hexyn-aws/install.ps1 | iex

Note: You will be prompted for a GitHub Personal Access Token (PAT) during installation.

Frequently Asked Questions

Why do I need a GitHub Token?

Since Hexyn AWS is hosted in a private repository, the installer requires a Personal Access Token (PAT) with read:packages or repo scope to download the official binary releases.

Which operating systems are supported?

We provide native binaries for Linux (AMD64/ARM64), macOS (Intel/Apple Silicon), and Windows (AMD64). All installations are self-contained.

How do I update to the latest version?

You can update directly from the CLI by running:

hexyn-aws update

Alternatively, you can re-run the installation command above.

Is it secure to use my token in the script?

Yes. The script only uses your token to authenticate with the GitHub API for the download. Your token is never stored or sent to any third-party servers.
