# Hexyn AWS Landing Page & Installer

This repository hosts the public landing page and installation scripts for **Hexyn AWS**, a high-performance AWS TUI tool.

## Purpose

- **Landing Page**: [flockyn.github.io/hexyn-aws](https://flockyn.github.io/hexyn-aws)
- **Unix Installer**: `install.sh`
- **Windows Installer**: `install.ps1`

## Repository Structure

- `index.html`: The main landing page built with Tailwind CSS.
- `install.sh`: Shell script for macOS/Linux installation.
- `install.ps1`: PowerShell script for Windows installation.

## Installation

Hexyn AWS is a public, open-source tool — no GitHub token is required to install or update it. The installers download the binary straight from the latest [public release](https://github.com/flockyn/hexyn-aws/releases).
